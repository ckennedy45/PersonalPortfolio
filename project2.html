Vision‑Based Waste Sorter

An embedded ML system that classifies recyclables on a conveyor in real time.

Overview

Trained a CNN to distinguish paper, plastic, glass, and landfill items and deployed it to a Raspberry Pi with a USB camera and stepper‑motor diverter.

Python PyTorch OpenCV Raspberry Pi
Demo Video Source Code

Model + Performance

MobileNetV3‑Small with transfer learning; 92% validation accuracy on custom dataset, 18 FPS on-device with TensorRT optimization.

TensorRT Data Augmentation Edge Inference

Impact

Demonstrated 35% improvement over manual sorting in student lab trials; designed modular pipeline for adding new classes without full retraining.
