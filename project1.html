Smart Task Tracker

A full-stack task management web app with real-time collaboration.

Overview

Built a Kanban-style task tracker supporting projects, columns, and tasks with drag-and-drop, @mentions, and activity feeds. Users can collaborate in real time using WebSockets.

TypeScript React Node.js WebSocket PostgreSQL
Live Demo Source Code

Key Features

- Real-time updates across clients using publish/subscribe.
- Role-based access control with JWT sessions.
- Optimistic UI updates and offline queueing.

JWT RBAC REST API

Impact

Reduced planning friction for student teams; handled 10k+ events/day in tests and sustained sub-100ms UI response for most interactions.
