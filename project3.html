Serverless Expense Analyzer

Ingests receipts, extracts data with OCR, and visualizes spending trends.

Overview

Implemented a pay‑as‑you‑go pipeline that accepts image or PDF receipts, runs OCR, and stores normalized transactions for dashboards and export.

TypeScript AWS Lambda API Gateway DynamoDB S3
Live Demo Source Code

Tech Highlights

Event‑driven architecture with S3 triggers; OCR via Textract; rate‑limited retries and dead‑letter queues; frontend dashboard built with vanilla JS modules.

Textract EventBridge DLQ

Impact

Processed 50k+ receipts in testing at <$3 total compute; simplified personal finance tracking with automatic category suggestions.
